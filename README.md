# basic llm chatbot

A simple app that shows how to make a chatbot for free using huggingface and streamlit. The app is live [here](https://basicchatbot-r9ttmvx3g2xskfd2fqhvdb.streamlit.app/).
You can also try different models, with a different number of parameters. This app uses only small language models, so it's energy consumption is far lower than the most famous ones. It also shows the user how much difference there is between the intelligence of a 1B parameters model and an 8B model.
